# Professor Posture Purrfect 

Professor Posture Purrfect is a cute macOS desktop reminder app that helps you improve your posture while working. A strict little black cat appears on your screen at regular intervals with playful reminders like:

> “Your shoulders are doing a sad shrimp pose.”  
> “Sit tall, human.”  
> “Uncurl your back, bestie.”

The goal is to make posture reminders feel fun, soft, and less annoying than typical productivity apps.

---

## Tech Stack

- **Tauri v2**: Desktop app framework
- **React**: Frontend UI
- **TypeScript**: Type-safe development
- **Vite**: Fast frontend tooling
- **Rust**: Tauri backend
- **Tauri Store Plugin**: Local preference storage
